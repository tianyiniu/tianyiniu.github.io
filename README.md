# Tianyi's Homepage

Website built on [*Academic Pages*](https://github.com/academicpages/academicpages.github.io), go check them out!

# Implementation Notes

1. Files uploaded into `files/` will appear at `https://tianyiniu.github.io/files/example.pdf`.
1. Check build status: repository settings --> "GitHub pages"

### Docker

After installing docker, build and execute the container by running:

```bash
docker compose up
```


### Live debugging

```bash
# Run
jekyll serve -l -H localhost

# or
./host_locally.sh

```

Access the website from `localhost:4000`
